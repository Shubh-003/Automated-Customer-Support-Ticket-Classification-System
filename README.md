
# Smart Support Desk

Enterprise AI-powered Ticket System using:

- Spring Boot 3
- Spring AI (Gemini)
- JWT + RBAC
- WebSockets (STOMP)
- PostgreSQL
- React Frontend
- Docker

## Windows Run Guide

1. Copy .env.example to .env and fill values
2. Run: docker-compose up --build
3. Backend: http://localhost:8080
