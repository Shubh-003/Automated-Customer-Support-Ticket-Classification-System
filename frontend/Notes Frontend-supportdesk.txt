## How will USER vs ADMIN/AGENT UI be different?
We will read the role from JWT login response.
Your backend already returns role inside the token (or user info).
Example login response:
{
  "accessToken": "...",
  "refreshToken": "...",
  "role": "ADMIN"
}

We store role in:
AuthContext

Example Auth State
{
  user: "shubh",
  role: "ADMIN",
  token: "..."
}


Phase 2: Authentication UI (Login + Register)
axiosClient.js:
 const axiosClient = axios.create({
               baseURL: "http://localhost:8080/api",
             })
store token at localstorage:  const token = localStorage.getItem("accessToken");

-> Now every request automatically attaches the JWT.

RegisterPage: usestte: usename and password then handleSubmit for axios.post("/auth/register")

CORS configuration in securityConfig.java- CorsConfigurationSource bean
🔐 Important Production Note:
Later in production we usually allow: https://yourdomain.com  not *.
Example: config.setAllowedOrigins(List.of("https://supportdesk.com"));
