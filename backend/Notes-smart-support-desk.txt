JWT Advance:
Login →
   Access Token (15 min)
   Refresh Token (7 days)
Access expires →
   Client calls /refresh →
   New Access Token generated
Logout →
   Refresh token revoked

# TARGET DESIGN for JWT:
 Access Token:
   15 minutes
   Used on every API call
 Refresh Token:
  Stored in DB
  7 days expiry
  Used only to generate new access token
